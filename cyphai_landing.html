How it works

  1. Verify — Ingest publicly available, signature‑verified posts.
  2. Understand — AI embeddings and reasoning grasp meaning beyond keywords.
  3. Rank & explain — Your trust graph and relevance scores surface better results, with clear “Why this result”.
For developers & reviewers
  • Vector search with Couchbase 7.6 FTS vector indexes (hybrid keyword + vector).
  • Trust/relevance scoring on GPUs using RAPIDS cuGraph / nx‑cuGraph (e.g., PageRank, Louvain, betweenness).
  • Model serving via NVIDIA Triton Inference Server (PyTorch/ONNX/TensorRT). Patched releases only.
  • Open‑weights LLMs for summaries, tagging, expertise inference; optional TensorRT‑LLM optimizations.
  • OpenShift + GitOps; cloud‑agnostic deployment.
